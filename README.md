## About Me

Hardware-focused engineer and researcher specializing in VLSI design, processor architecture, and FPGA-based systems. Currently pursuing advanced studies in VLSI at IIT Mandi, with a focus on building secure, efficient, and observable hardware systems.

**Core Focus Areas:**

- Embedded systems and IoT infrastructures
- Neuromorphic computing architectures
- RF signal processing and sensing systems
- Digital design, verification, and formal methods

## What I'm Building

I design hardware-first solutions that emphasize performance, security, and observability. Current work includes developing timing-based hardware security mechanisms, building custom RISC-V cores with specialized execution units, and prototyping neuromorphic computing architectures on FPGAs.

**Engineering Philosophy:**
- Hardware-first thinking — optimize at the RTL level
- Performance-aware design — cycles and area matter
- Security-aware compute — defense in silicon
- Observable and testable systems — verification is non-negotiable

## Selected Projects

### 🔒 **Execution Time Signatures (ETS)**
Hardware-based timing attack detection system that identifies side-channel vulnerabilities through microarchitectural execution time analysis. Built custom RISC-V instrumentation for runtime security monitoring.

### ⚡ **RISC-V Processor Implementations**
Designed and verified multiple RISC-V core variants including single-cycle, pipelined, and out-of-order execution models. Implemented custom branch predictors, cache hierarchies, and specialized execution units.

### 🧠 **Neuromorphic RISC-V Experiments**
Exploring neuromorphic computing primitives on RISC-V architectures. Prototyped event-driven processing units and spike-based communication protocols on FPGA.

### 📡 **SmartRF Signal Classification Pipeline**
End-to-end RF signal processing and classification system. Developed real-time feature extraction algorithms and deployed on embedded edge devices for spectrum sensing applications.

### 🌐 **Embedded IoT Telemetry Systems**
Architected low-power sensor networks with multi-protocol support (UART, SPI, I2C, LoRa, LTE). Built custom firmware for STM32 and ESP32 platforms with focus on reliability and energy efficiency.

### 🔬 **CMOS Analog & Digital Design**
Full-custom CMOS circuit design including operational amplifiers, ADCs, and standard cell libraries. Performed layout, DRC/LVS verification, and post-layout simulation using Cadence and Tanner EDA tools.
