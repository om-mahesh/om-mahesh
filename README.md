## 🚀 About Me

I'm a **Hardware Engineer** and **VLSI Researcher** at **IIT Mandi**, specializing in processor architecture, FPGA-based systems, and hardware security. My work focuses on building **observable, efficient, and secure** computing systems from the ground up — because **hardware is truth**.

- 🔭 Currently working on **[Efficient FPGA-Based Phase Unwrapping](https://github.com/om-mahesh/Efficient-FPGA-Based-Parallel-Phase-Unwrapping-Hardware-Architecture)**
- 🧠 Researching **Neuromorphic RISC-V architectures** and event-driven computing
- 🔒 Building **hardware-based timing attack detection** systems
- 📡 Developing **RF signal processing** and classification pipelines
- 💡 Passionate about **open-source hardware** and **RISC-V ecosystem**
- 🎯 Core belief: **Hardware-first thinking, security-aware design**

## 💼 Featured Projects

### 🔒 [Hardware-Based Execution Time Signatures (ETS)](https://github.com/om-mahesh/Hardware-Based-Execution-Time-Signatures-ETS-for-Real-Time-Detection-of-Timing-Attacks)
**Real-time timing attack detection system**  
Built custom RISC-V instrumentation for microarchitectural timing analysis. Detects side-channel vulnerabilities through execution time profiling.  
`C` `Hardware Security` `Side-Channel Defense`

### ⚡ [Branch Prediction Unit with Optimized FSM](https://github.com/om-mahesh/Branch-Prediction-Unit-with-Temporal-Properties-and-Optimized-FSM)
**Advanced branch predictor with temporal properties**  
Designed and verified branch prediction unit with optimized finite state machine for improved pipeline performance.  
`Verilog` `Computer Architecture` `⭐ 1`

### 📡 [FPGA-Based Phase Unwrapping Architecture](https://github.com/om-mahesh/Efficient-FPGA-Based-Parallel-Phase-Unwrapping-Hardware-Architecture)
**High-throughput parallel pipeline for digital holographic microscopy**  
Non-iterative DCT-based architecture implementing Laplacian → DCT → Frequency Processing → IDCT stages for fixed, high-throughput phase unwrapping.  
`VHDL` `FPGA` `Signal Processing`

### 🧠 [Neuromorphic RISC-V](https://github.com/om-mahesh/neuromorphic-riscv)
**Event-driven computing experiments on RISC-V**  
Exploring neuromorphic computing primitives and spike-based communication protocols on RISC-V architectures.  
`RISC-V` `Neuromorphic` `Hardware Acceleration`

###  ️ [Power-Aware LBIST for MIPS32](https://github.com/om-mahesh/Power-Aware-LBIST-for-a-MIPS32-Single-Cycle-Core)
**Built-in self-test system for processor cores**  
Logic Built-In Self-Test (LBIST) implementation with power-aware testing strategies for MIPS32 single-cycle architecture.  
`VHDL` `DFT` `Processor Testing`

### 🔬 [Telescopic CMOS Op-Amp (180nm)](https://github.com/om-mahesh/Design-of-a-High-Speed-Low-Noise-Telescopic-CMOS-Operational-Amplifier-in-180-nm-Process)
**High-speed low-noise analog design**  
Full-custom CMOS operational amplifier design with layout, DRC/LVS verification, and post-layout simulation.  
`Analog Design` `CMOS` `Cadence/Tanner`

###   [SmartRF Signal Classification](https://github.com/om-mahesh/SmartRF)
**Wireless signal intelligence with deep learning**  
End-to-end RF signal processing pipeline with real-time feature extraction and ML-based classification for spectrum sensing.  
`Signal Processing` `Deep Learning` `RF Systems`

### 🔧 [Omduino X1 – Custom Arduino Board](https://github.com/om-mahesh/Omduini-x1)

## 🎯 Engineering Philosophy

> **Hardware is truth. Software is proof.**

- 🔹 **Hardware-first thinking** — Optimize at the RTL level, not as an afterthought
- 🔹 **Performance-aware design** — Every cycle and every gate matters
- 🔹 **Security-aware compute** — Defense must be built into silicon
- 🔹 **Observable systems** — What you can't measure, you can't trust
- 🔹 **Verifiable correctness** — Simulation is necessary, formal verification is sufficient
